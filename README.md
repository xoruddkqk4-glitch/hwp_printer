# HWP / HWPX 일괄 인쇄 유틸리티 (Batch Printer Tool)

이 프로그램은 지정된 폴더 내의 한글 파일(`*.hwp`, `*.hwpx`)들을 탐색하여 한컴오피스 프로그램 제어(COM Automation) 및 윈도우 프린터 설정 제어를 통해 한 번에 일괄 인쇄할 수 있도록 돕는 유틸리티입니다.

단면 인쇄 및 양면 인쇄(긴 쪽/짧은 쪽 넘김) 옵션을 간편하게 설정하여 출력할 수 있습니다.

---

## 🛠️ 준비 사항 및 설치 방법

이 프로그램을 실행하기 위해서는 아래의 요구 사항이 필요합니다.

1. **한컴오피스 한글 설치**: 스크립트 실행 컴퓨터에 정식 한글 프로그램이 설치되어 있어야 합니다. (백그라운드에서 한글 객체를 생성해 제어합니다.)
2. **Python 3.x 설치**: Python이 설치되어 있어야 합니다.
3. **pywin32 라이브러리 설치**: 윈도우 API 및 한글 COM 객체를 다루기 위해 필요한 필수 라이브러리입니다.
   - 명령 프롬프트(CMD) 또는 터미널을 열고 아래 명령어를 실행하여 설치합니다.
   ```bash
   pip install pywin32
   ```

## 🚀 사용 방법

### 1. GUI 모드로 실행 (권장)
가장 편리하게 시각적으로 폴더를 선택하고 파일을 개별 체크하여 인쇄하는 방법입니다. 인자값 없이 프로그램을 실행하면 GUI 모드로 켜집니다.

```bash
python print_hwp_tool.py
```

* **동작 순서**:
  1. 프로그램 실행 후 우측 상단의 `폴더 선택 🔍` 버튼을 누르고 한글 파일들이 모여있는 폴더를 지정합니다.
  2. 폴더 내 한글 파일 목록이 표 형식으로 불러와지며, 인쇄하고 싶지 않은 파일은 체크박스를 클릭해 해제할 수 있습니다.
  3. 우측 설정 카드에서 출력할 **프린터**, **인쇄 부수**, **인쇄 모드(기본설정 / 단면 / 양면-긴쪽 / 양면-짧은쪽)**를 선택합니다.
  4. `인쇄 시작 ▶` 버튼을 클릭하면, 하단 진행 로그 창에 상세 스텝이 표시되며 인쇄가 순차적으로 실행됩니다.

### 2. CLI 모드로 실행 (명령줄 일괄 처리)
백그라운드 작업 스크립트나 배치 파일 등에서 GUI 없이 한 번에 인쇄하려는 경우 사용합니다.

```bash
# 기본 사용법 (기본 프린터, 기본 설정 사용)
python print_hwp_tool.py --dir "C:\Users\user\Documents\TestFolder"

# 양면 인쇄(긴 쪽 넘김), 2부씩 인쇄
python print_hwp_tool.py --dir "C:\Users\user\Documents\TestFolder" --duplex long --copies 2

# 특정 프린터 지정, 단면 인쇄
python print_hwp_tool.py --dir "C:\Users\user\Documents\TestFolder" --printer "HP OfficeJet 8710" --duplex simplex
```

* **CLI 옵션 설명**:
  - `--dir [경로]`: 한글 파일이 위치한 폴더 경로 (필수)
  - `--printer "[프린터명]"`: 사용할 프린터 명칭 (생략 시 기본 프린터 사용)
  - `--duplex [default|simplex|long|short]`: 단면/양면 설정 (생략 시 기본 프린터 기본값 적용)
  - `--copies [숫자]`: 인쇄 부수 (생략 시 1부)

## 🔒 중요: 한글 API 보안 승인 팝업 해결 방법

처음 프로그램을 실행하고 인쇄를 진행할 때, 한글 프로그램이 백그라운드에서 열리면서 **"외부 애플리케이션이 한글 문서에 접근하려고 합니다. 허용하시겠습니까?"**와 같은 보안 승인 안내창이 파일마다 반복해서 뜰 수 있습니다. 

이를 방지하기 위해 한컴오피스 공식 API 보안 모듈을 설치하거나 레지스트리를 설정해야 합니다.

### 해결 방법 1: 한글 기본 보안 설정 완화 (가장 간단)
1. 한글 프로그램을 개별적으로 실행합니다.
2. 상단 메뉴에서 **[도구] ➡️ [스크립트 매크로] ➡️ [매크로 보안 설정]**으로 이동합니다.
3. 보안 수준을 **"낮음"**으로 설정하고 설정을 완료합니다. (보안 수준을 낮추면 자동화 API 사용 시 경고 팝업이 나타나지 않습니다.)

### 해결 방법 2: 보안 승인 모듈(DLL) 등록
한글 보안 설정을 "낮음"으로 바꾸지 않고 유지하면서 안전하게 자동화를 실행하려면 보안 DLL 파일을 등록해야 합니다.
1. `FilePathCheckDLL.dll` 파일을 인터넷 또는 한컴디벨로퍼 포럼 등에서 다운로드합니다. (한컴오피스 기본 설치 경로에도 종종 포함되어 있습니다.)
2. 다운로드한 DLL 파일을 안전한 경로(예: `C:\HwpAutomation\FilePathCheckDLL.dll`)에 배치합니다.
3. **윈도우 레지스트리 에디터 (`regedit`)**를 실행합니다.
4. 아래 경로로 이동합니다 (HwpAutomation 폴더가 없다면 마우스 우클릭으로 키를 새로 생성합니다):
   - `HKEY_CURRENT_USER\Software\HNC\HwpAutomation\Modules`
5. `Modules` 키 폴더 안에서 마우스 우클릭 ➡️ **[새로 만들기] ➡️ [문자열 값(REG_SZ)]**을 선택합니다.
   - 값 이름: `FilePathCheckerModule` (원하는 이름 아무거나)
   - 값 데이터: DLL의 절대 경로 (예: `C:\HwpAutomation\FilePathCheckDLL.dll`)
6. 설정을 완료하면 본 프로그램이 한글을 제어할 때 경고 팝업 없이 부드럽게 대량 인쇄를 처리할 수 있게 됩니다.
